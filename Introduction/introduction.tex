\section{Motivations}
\label{motivations}

\begin{itemize}
    \item \textbf{Driver:} Cloud adoption offering cost savings, scalability, and access to advanced capabilities for SMEs.
    \item \textbf{Trend:} Increasing shift in developing regions to bypass infrastructure gaps.
    \item \textbf{Issue:} Benefits accompanied by security risks (insecure defaults, weak access controls).
    \item \textbf{Constraint:} SMEs lack technical expertise, budget, and awareness of cyber risks.
    \item \textbf{Problem with Standards:} Frameworks like NIST and ISO are designed for large organizations with mature governance.
    \item \textbf{Need:} A lightweight, deployment-focused framework sensitive to SME constraints.
\end{itemize}

\section{Context of the Study}

    \item \textbf{Region:} Middle East, Africa, South Asia (MENA/SSA).

\section{Objectives and Contributions}
\label{objectives}

\textbf{Primary Objective:} Design and validate a Secure Cloud Deployment Framework (SCDF) tailored to SME constraints in developing regions.

\textbf{Specific Goals:}
\begin{itemize}
    \item Analyze recurring security challenges for SMEs.
    \item Review existing standards (applicability vs. limitations).
    \item Propose a modular, open-source-oriented deployment model.
    \item Evaluate feasibility via simulated testing and expert review.
\end{itemize}

\textbf{Contributions:}
\begin{enumerate}
    \item Context-aware framework bridging theory and practice.
    \item Comparative perspective on regional cybersecurity readiness (Lebanon, MENA, SSA).
    \item Practical insights for policymakers and SME decision-makers.
\end{enumerate}
