## 1. Zulkifli2023ProposedMFA
**Article:** A Proposed Multifactor Authentication Framework for SME in Cloud Computing Environment

| Column | Extraction |
| :--- | :--- |
| **Study Context** | **Geographic:** Malaysia (implied by authors' affiliation with UTM/UiTM).<br>**Industry:** SMEs (General).<br>**Methodology:** Literature Review and Conceptual Framework Proposal.<br>**Period:** Published 2023. |
| **Cloud Adoption Barriers** | **Security:** Data security, identity theft, unauthorized access, password vulnerabilities (weak, shared).<br>**Technical:** Vulnerability of single-factor authentication (SFA). |

| **Frameworks and Solutions** | **Proposed:** A new Multifactor Authentication (MFA) Framework for SMEs.<br>**Features:** Combines Knowledge (Password), Possession (Smart Card/Mobile), and Biometric factors.<br>**Type:** Conceptual Security Framework.<br>**Target:** Specifically designed for SMEs to enhance cloud security. |
| **Evaluation Methods** | **Methodology:** Conceptual proposal (Framework design).<br>**Validation:** Future work includes measuring the effectiveness and efficiency of the proposed framework. |
| **Key Findings** | **Findings:** SFA is insufficient for cloud security. MFA provides stronger security but must be balanced with usability for SMEs.<br>**Gap:** Existing authentication methods may be too complex or costly for SMEs.<br>**Recommendation:** Implement the proposed MFA framework to mitigate unauthorized access risks in cloud environments. |

## 2. Vuong2015Towards
**Article:** Towards efficient and secure data storage in multi-tenant cloud-based CRM solutions

| Column | Extraction |
| :--- | :--- |
| **Study Context** | **Geographic:** Germany (implied by authors' affiliation with CAS Software AG).<br>**Industry:** CRM Software Vendors / SMEs using CRM.<br>**Methodology:** Case Study / Technical Implementation.<br>**Period:** Published 2015. |
| **Cloud Adoption Barriers** | **Security:** Data Confidentiality, Integrity, Isolation in Multi-tenant environments.<br>**Regulatory:** Compliance with data protection laws.<br>**Technical:** Performance overhead of encryption. |

## 3. Palanisamy2020Users
**Article:** Usersâ€™ attitude on perceived security of mobile cloud computing: empirical evidence from SME users in China

| Column | Extraction |
| :--- | :--- |
| **Study Context** | **Geographic:** China.<br>**Industry:** SMEs (Various sectors).<br>**Methodology:** Survey (Empirical Study based on TAM/TPB).<br>**Sample:** 316 valid responses from SME employees.<br>**Period:** Published 2017/2020. |
| **Cloud Adoption Barriers** | **Security:** Perceived security risks (Privacy, Data Loss) are the primary barrier.<br>**Technical:** Perceived ease of use affects adoption. |
| **Local Constraints** | **Cultural:** Trust in service providers is a significant factor in China.<br>**Expertise:** Technical complexity perceives security risk. |
| **Frameworks and Solutions** | **Model:** Theoretical model integrating TAM (Technology Acceptance Model) with Perceived Security and Trust.<br>**Findings:** Trust negatively affects perceived security risk; perceived security positively affects attitude. |
| **Evaluation Methods** | **Methodology:** Structural Equation Modeling (SEM).<br>**Metrics:** Path coefficients, R-squared values for Attitude and Intention. |
| **Key Findings** | **Findings:** Perceived security is the most significant predictor of attitude towards mobile cloud computing. Trust reduces perceived risk.<br>**Implication:** Cloud providers must focus on building trust and demonstrating security to win over Chinese SMEs. |

## 4. Asiaei2019Multifaceted
**Article:** A multifaceted framework for adoption of cloud computing in Malaysian SMEs

| Column | Extraction |
| :--- | :--- |
| **Study Context** | **Geographic:** Malaysia.<br>**Industry:** SMEs (Manufacturing and Services).<br>**Methodology:** Quantitative Survey.<br>**Sample:** 324 SME managers/owners.<br>**Period:** Published 2019. |
| **Cloud Adoption Barriers** | **Security:** Security and Privacy concerns are significant but less than expected.<br>**Organizational:** Top management support, Technology readiness.<br>**Environmental:** Competitive pressure, Trading partner support. |
| **Local Constraints** | **Infrastructure:** Broadband quality in some areas.<br>**Government:** Government support is a key moderator in Malaysia. |
| **Frameworks and Solutions** | **framework:** TOE (Technology-Organization-Environment) framework extended with DOI (Diffusion of Innovation).<br>**Features:** Identifies key determinants of adoption. |
| **Evaluation Methods** | **Methodology:** PLS-SEM (Partial Least Squares Structural Equation Modeling).<br>**Metrics:** H-hypothesis testing significance. |
| **Key Findings** | **Findings:** Relative advantage, compatibility, and top management support are strongest predictors. Security concern was found to be insignificant in this specific study (contrary to others).<br>**Success Factor:** Government support positively moderates the relationship between organizational factors and adoption. |

## 5. Fen2024Cloud
**Article:** Cloud Computing Adoption Among Malaysian SMEs Manufacturers: The Role of Relative Advantage, Complexity and Cybersecurity Readiness

| Column | Extraction |
| :--- | :--- |
| **Study Context** | **Geographic:** Malaysia (Penang).<br>**Industry:** Manufacturing SMEs.<br>**Methodology:** Quantitative Survey.<br>**Sample:** 137 manufacturing SMEs.<br>**Period:** Published 2024. |
| **Cloud Adoption Barriers** | **Technical:** Complexity of cloud solutions.<br>**Security:** Cybersecurity readiness is a critical factor. |
| **Local Constraints** | **Sector-specific:** Manufacturing SMEs face unique integration challenges with legacy machinery (Industry 4.0).<br>**Expertise:** Lack of cybersecurity skills. |
| **Frameworks and Solutions** | **Model:** Extended TOE framework focusing on Technological factors (Relative Advantage, Complexity, Cybersecurity Readiness). |
| **Evaluation Methods** | **Methodology:** Multiple Linear Regression (SPSS).<br>**Metrics:** Beta coefficients, Significance levels. |
| **Key Findings** | **Findings:** Relative Advantage and Cybersecurity Readiness have a significant positive relationship with Cloud Adoption. Complexity has a significant negative relationship.<br>**Implication:** Simplifying cloud solutions and enhancing cybersecurity readiness are crucial for Malaysian manufacturers. |

## 6. Akram2017Innovation

| Column | Extraction |
| :--- | :--- |
| **Study Context** | **Geographic:** Pakistan.<br>**Industry:** SMEs (Textile, IT, Food, etc.).<br>**Methodology:** Quantitative Survey.<br>**Sample:** 289 SMEs.<br>**Development:** Developing country.<br>**Period:** Published 2017. |
| **Cloud Adoption Barriers** | **Security:** Security concerns are a major barrier.<br>**Technical:** Complexity, Compatibility.<br>**Cost:** Cost is a significant driver (Relative Advantage). |
